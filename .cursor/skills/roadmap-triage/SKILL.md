# Roadmap Triage

## Goal

Convert a roadmap initiative into delivery-ready feature slices.

## Workflow

1. Restate the outcome metric and kill criteria.
2. Identify dependencies, risks, and unknowns.
3. Split work into increments: work/right/good/fast phases.
4. Produce implementation slices with acceptance tests.
5. Mark required stage gates and owners.
